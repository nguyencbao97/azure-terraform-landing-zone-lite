\# Azure Terraform Landing Zone Lite



This is my personal Azure Terraform lab. I built it to get more hands-on practice with Terraform, Azure infrastructure, and GitHub Actions.



This project helps demonstrate my hands-on practice with Terraform, Azure infrastructure, cloud security basics, monitoring, and CI/CD validation.



\## Why I Built This



I currently work with Azure and Microsoft 365, but I cannot freely test Terraform deployments in my work environment. I made this project so I could practice infrastructure-as-code in a safe personal lab.



\## What This Project Includes



\* Azure resource group

\* Virtual network

\* Public and private subnets

\* Network security groups

\* Storage account

\* Key Vault

\* Log Analytics workspace

\* Optional Linux App Service

\* Application Insights

\* Managed identity

\* RBAC role assignment

\* GitHub Actions Terraform validation



\## What I Practiced



\* Writing Terraform configuration files

\* Using Terraform modules

\* Building basic Azure networking

\* Adding security resources like Key Vault and NSGs

\* Using managed identity and RBAC

\* Adding monitoring with Log Analytics and Application Insights

\* Running Terraform format and validation checks with GitHub Actions



\## Project Status



The project files are uploaded to GitHub, and the GitHub Actions workflow passed successfully.



\## Security Note



This repo only uses example values. I did not upload real secrets, keys, state files, or production information.



\## Notes



This is a personal learning project, not a production environment. I kept it simple so I can understand each part and explain it clearly in interviews.
